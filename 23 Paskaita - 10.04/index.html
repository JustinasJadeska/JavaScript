Masyvų iteraciniai metodai

Iteraciniai masyvų metodai yra tokie metodai, kur sukant ciklą per masyvą, galima kažką atlikti su kiekvienu to masyvo elementu.

ForEach

Syntax:

masyvas.forEach((el)=>{ veiksmai... }) masyvas.forEach((el, i)=>{ veiksmai... }) masyvas.forEach((el, i, arr)=>{ veiksmai... })

ForEach metodas - suka ciklą per visą masyvą ir kiekvienos iteracijos metu atlieka kažkokius veiksmus su tos iteracijos masyvo elementu. ForEach'as negali grąžinti reikšmės (neturi return). ForEach'as negali būti sustabdytas (neveikia break).

Filter

Syntax:

masyvas.filter((el)=>{ veiksmai... sąlyga }) masyvas.filter((el, i)=>{ veiksmai... sąlyga }) masyvas.filter((el, i, arr)=>{ veiksmai... sąlyga })

Filter metodas - suka ciklą per visą masyvą ir kiekvienos iteracijos metu tikrina ar iteracijos masyvo elementas atitinka sąlyga. Jeigu sąlyga grąžina tiesą - tas elementas yra grąžinamas kaip naujo masyvo dalis; Jeigu sąlyga grąžina melą - tas elementas nėra grąžinamas kaip naujo masyvo dalis.

Map

Syntax:

masyvas.map((el)=>{ veiksmai... naujasPavidalas }) masyvas.map((el, i)=>{ veiksmai... naujasPavidalas }) masyvas.map((el, i, arr)=>{ veiksmai... naujasPavidalas })

Map metodas - suka ciklą per visą masyvą ir kiekvienos iteracijos metu kažkaip mutuoja iteracijos masyvo elementą ir grąžina jį nauju pavidalu.

Reduce

Syntax:

masyvas.reduce((acc, curr) => { veiksmai... }) masyvas.reduce((acc, curr, i) => { veiksmai... }) masyvas.reduce((acc, curr, i, arr) => { veiksmai... }) masyvas.reduce((acc, curr) => { veiksmai... }, initialValue) masyvas.reduce((acc, curr, i) => { veiksmai... }, initialValue) masyvas.reduce((acc, curr, i, arr) => { veiksmai... }, initialValue)

Reduce metodas - suka ciklą per visą masyvą ir kiekvienos iteracijos metu su akumuliatoriumi(acc) (reikšmė, kurią grąžino praeitas ciklas) atlieka kažkokį veiksmą su dabartinės iteracijos masyvo elementu (curr). Metodo gale - grąžina vieną reikšmę, kuri yra kažkoks junginys visų reikšmių. Jeigu yra nurodyta pradinė reikšmė (initialValue), tuomet pirmosios iteracijos metu akumuliatorius(acc) yra pradinė reikšmė(initialValue), o pirmasis masyvo elementas yra dabartinė reikšmė(curr).

ReduceRight

Lygiai tas pats kas ir reduce, tik kad iš kito galo (right to left (rtl)).

FlatMap

Apjungtas flat(1) ir map metodai. Neturi galimybės keisti flat'o lygmens.

Some

Some metodas - suka ciklą per visą masyvą ir kiekvienos iteracijos metu tikrina ar bent vienas iteracijos masyvo elementas atitinka nurodytą sąlygą. Jeigu bent vienas elementas atitinka nurodytą sąlygą - metodas grąžina true. Jeigu nei vienas elementas neatitinka nurodytos sąlygos - metodas grąžina false.

Every

Every metodas - suka ciklą per visą masyvą ir kiekvienos iteracijos metu tikrina ar kiekvienas vienas iteracijos masyvo elementas atitinka nurodytą sąlygą. Jeigu kiekvienas elementas atitinka nurodytą sąlygą - metodas grąžina true. Jeigu bent vienas elementas neatitinka nurodytos sąlygos - metodas grąžina false.

Find

Find metodas - suka ciklą per visą masyvą ir kiekvienos iteracijos metu tikrina ar iteracijos masyvo elementas atitinka nurodytą sąlygą. Jeigu iteracijos masyvo elementas atitinka nurodytą sąlygą - yra grąžinamas tas elementas. Jeigu nei vienas iteracijos masyvo elementas neatitinka nurodytos sąlygos - grąžinamas undefined.

FindLast

FindLast metodas daro tą patį ką Find tik nuo kito galo (right to left (rtl)).

FindIndex

FindIndex metodas - suka ciklą per visą masyvą ir kiekvienos iteracijos metu tikrina ar iteracijos masyvo elementas atitinka nurodytą sąlygą. Jeigu iteracijos masyvo elementas atitinka nurodytą sąlygą - yra grąžinamas to elemento indeksas. Jeigu nei vienas iteracijos masyvo elementas neatitinka nurodytos sąlygos - grąžinamas -1.

FindLastIndex

FindLastIndex - daro tą patį ką ir findIndex, tik kad iš kito galo (right to left (rtl)).
