API

Application Programming Interface (API) - pateiktą užklausą iš aplikacijos (front-end) išsiunčia į serverį (back-end) ir jeigu viskas tvarkoje - grąžina duomenis pagal pateiktą užklausą, jeigu kažkas negerai - grąžina atitinkamą error žinutę.

Fetch

Komanda/funckija/metodas, kurio pagalba mes galime siųsti užklausas į serverį.

Fetch veikimo eiga

  1. Fetch'inti (siūsti užklausą) duomenis, naudojant metodus:
    • GET - gauti informaciją. Default metodas.
    • POST - pateikti informaciją
    • DELETE - trinti informaciją
    • PATCH - redaguoti informaciją
    • PUT - redaguoti informaciją
  2. Apdoroti gautą informaciją
  3. Naudoti gautą informaciją
  4. Pagaunam (catch) ir kažką atliekam su error (vykdosi tik error atveju)
  5. Galiausiai (finally) dar kažką atliekam (vykdosi bet kokiu atveju)
