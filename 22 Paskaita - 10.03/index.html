Array

Savybės/Properties:

  • arejus.length - grąžino masyvo ilgį (iš kelių elementų masyvas yra sudarytas).

Metodai:

  • arejus.at(number) - grąžina masyvo number elementą.
  • arejus.concat(arejus1, arejus2...) - grąžina masyvų junginį.
  • arejus.fill(reiksme, nuo, iki) - užpildo masyvą reiksme'mis nuo nurodyto indekso iki nurodyto indekso. MODIFIKUOJA pradinį masyvą.
  • arejus.includes(reiksme) - tikrina ar bent vienas masyvo elementas yra lygus reiksme'i.
  • arejus.indexOf(reiksme, fromIndex) - grąžina kurioje pozicijoje masyve yra nurodyta reiksme. Galima pasirinkti nuo kurios pozicijos ieškoti. Jeigu neranda - grąžina -1.
  • arejus.lastIndexOf(reiksme, fromIndex) - grąžina kurioje pozicijoje masyve yra nurodyta reiksme. Ieško nuo galo judėdamas į pradžią. Galima pasirinkti nuo kurios pozicijos ieškoti. Jeigu neranda - grąžina -1.
  • arejus.join(skirtukas) - sujungia masyvą į string'ą, tarp masyvo elementų sudėdamas skirtukus. Grąžina naujai sugeneruotą string'ą.
  • arejus.toString() - tas pats kas join, tik be galimybės nusirodyti skirtuką.
  • arejus.pop() - išima reikšmę iš masyvo galo ir ją grąžina. MODIFIKUOJA pradinį masyvą.
  • arejus.push(reiksme) - prideda reikšmę į masyvo galą ir grąžina naują masyvo ilgį. MODIFIKUOJA pradinį masyvą.
  • arejus.shift() - išima reikšmę iš masyvo pradžios ir ją grąžina. MODIFIKUOJA pradinį masyvą.
  • arejus.unshift(reiksme) - prideda reikšmę į masyvo pradžią ir grąžina naują masyvo ilgį. MODIFIKUOJA pradinį masyvą.
  • arejus.reverse() - apsuka masyvą. MODIFIKUOJA pradinį masyvą.
  • arejus.toReversed() - apsuka masyvą. NEmodifikuoja pradinio masyvo.
  • arejus.slice(nuo, iki) - grąžina elementus nuo(imtiniai), iki(neimtinai) pozicijos.
  • arejus.splice(nuo, kiek, el1, el2, el3...) - iš pradinio masyvo, išima nuo nurodytos pozicijos, nurodytą kiekį elementų ir į tą vietą patalpina naujus išvardintus elementus. Grąžina išimtus elementus. MODIFIKUOJA pradinį masyvą.
  • arejus.toSpliced(...) - tas pats kas splice, tik kad NEmodifikuoja pradinio masyvo.
  • arejus.sort(palyginimofunc) - grąžina surikiuotą masyvą nurodyta tvarka.MODIFIKUOJA pradinį masyvą. dažniausia funkcija skaičiams: (a,b) => a-b dažniausia funkcija žodžiams: (a,b) => a.localeCompare(b)
  • arejus.toSorted(...) - tas pats kas sort, tik kad NEmodifikuoja pradinio masyvo.
  • arejus.with(index, reiksme) - grąžina naują masyvą su pakeista reiksme nurodytame indekse. NEmodifikuoja pradinio masyvo.

Funkcijos

Funkcija - kažkoks parašytas kodas, kurį galima iškviesti bet kuriuo momentu ir bet kiek kartų. Funkcijos kodas nesivykdo kol ji nėra iškviesta.

Funkcijos aprašymas: function f_Vardas(sk1, sk2...){veiksmai} Funkcijos iškvietimas: f_Vardas(arg1, arg2...);

Parametrai ir argumentai

Parametrai - funkcijos viduje naudojami duomenys, kurie yra gaunami funkcijos iškvietimo metu.

Argumentai - perduodami duomenys kreipimosi į funkciją metu.

Callbakc

Callback - iškvietimas ir grįžtamasis ryšys. Kai funkcija arba metodas, kreipiasi į kitą funkciją.

Anonymous/Nameless Function / Anoniminės/Bevardės Funkcijos

Funkcijos, kurioms nėra priskirtas vardas. Kitaip pasakius, tai funkcijos, kurios egzistuoja tik kaip koks nors callback'as.

Arrow Function / Lambda

ES6 versijoje išleistas naujas bevardės funkcijos aprašymo būdas.

Norint sukurti vardinę funkciją naudojant arrow function, reikia sukurti kintamąjį, kuriam prilyginsi tą bevardę funkciją.

Arrow function užrašymo būdai

  • (par1, par2, par3...) => { veiksmai... }
  • par => { veiksmai... }
  • (par1, par2, par3...) => veiksmas
  • par => veiksmas
