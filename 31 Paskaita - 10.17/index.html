Duomenų saugojimas vartotojo dalyje

Cookies, Session Storage ir Local Storage informacija yra saugojama jūsų įrenginyje/aplikacijoje/naršyklėje...
Skirtingose svetainėse ir skirtingose naršyklėse Cookies/Session/Local Storage yra saugojama atskirai.

Cookies

Ne sausainiai, bet duomenys saugojami svetainėje.

Saugojama iki 4kb.
Reikia rankiniu būdu nustatyti iki kada duomenys yra saugojami.
Siunčiami kartu su request'ais į back-end'ą.

Session Storage

Laiko duomenis aplikacijoje/naršyklėje, per kurią atidaroma kažkuri svetainė, kuriai tie duomenys priskiriami.
Kai pasibaigia sesija (kai yra išjungiama svetainė) šitie duomenys yra ištrinami.

Storage

  • sessionStorage.setItem('raktas', 'reiksme')
  • sessionStorage.getItem('raktas')
  • sessionStorage.removeItem('raktas')
  • sessionStorage.clear()
  • sessionStorage.key(index)
  • sessionStorage.length

Local Storage

Laiko duomenis aplikacijoje/naršyklėje, per kurią atidaroma kažkuri svetainė, kuriai tie duomenys priskiriami.
Duomenys išsisaugoja per daugelį sesijų. Ištrinami gali būti tik naudojant atitinkamus metodus arba per dev tools.

Storage

  • localStorage.setItem('raktas', 'reiksme')
  • localStorage.getItem('raktas')
  • localStorage.removeItem('raktas')
  • localStorage.clear()
  • localStorage.key(index)
  • localStorage.length

JavaScript Object Notation (JSON)

JSON yra JavaScript objektų aprašymas string'ų pavidalu.
JSON viduje viskas rašoma kaip kad būtų rašomas objektas, BET:
1) JSON viduje negali būti metodų. Jeigu metodas yra, jis yra panaikinamas.
2) JSON viduje visi žodžiai apart skaičių ir boolean'ų turi būti string'ai su dvigubomis kabutėmis.

Objektą galima paversti į JSON sintaksę arba JSON sintaksę galima paversti į Objektą naudojant atitinkamus metodus.

JSON Metodai

  • JSON.stringify(objektas)
  • JSON.parse(JSON_stringas)
