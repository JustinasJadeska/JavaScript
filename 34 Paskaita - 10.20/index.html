JSON server

JSON serveris yra Node Package Manager (NPM) paketas, kurio pagalba galima susikurti lokalų API serverį su savo duomenimis, kurie privalo būti JSON formatu.

Norint naudotis JSON serveriu, reikia: npm (įsirašo su NODE), JSON failo atitinkamu formatu parašyto, įsirašyti json-server globaliai, pasileisti JSON failą su json-server.

Terminalo komandos

  • cd- change directory - naviguotis direktorijose
  • ls - list files - išvardina elementus esamojoje direktorijoje
  • json-server --watch fileName.json
