HTML formų validavimas

Formų validavimas yra reikalingas norint nustatyti kažkokius reikalavimus formos laukams (kitaip pasakius, įvesčių reikalavimus).
Formų validacija neleidžia vartotojui pateikti formos, jeigu bent vienas iš formos laukų neatitinka nurodytos validacijos.

  • required
  • minlength - text/password/url/email
  • maxlength - text/password/url/email
  • min - number/date/range
  • max - number/date/range
  • step - number/range/date
  • pattern - password/tel
  • accept - file
  • multiple - file/email

Document Object Model

Node

Node'as - HTML elementas JS'o aplinkoje.

Daug geresnis būdas kurti HTML elementus per JS yra ne užpildant tėvinio elemento innerHTML'ą ar textContent'ą, bet kuriant Node'us kaip kintamuosius ir priskiriant juos vienus kitiems.

Node'ų metodai

  • createElement('tagVardas')
  • createTextNode('tekstas...')
  • kam.append(ką) - galima betką ir betkiek
  • kam.appendChild(ką) - galima TIK Node'ą
  • kam.setAttribute('name','value')
  • classList
    • add('klasesVardas','kitaKlase', ...)
    • remove('klasesVardas', 'kitaKlase', ...)
    • replace('klasesVardas', 'kitaKlase')
    • toggle('klaseViena')
